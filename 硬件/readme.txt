Summry and reflection:

	使用嘉立创eda专业版打开
	从投板到收到pcb总共时间大概在5天-7天
	画板子原理图一定要仔细，要去看参考手册，读它的sheet和参考电路原理图和建议布板方式。
	先画芯片的外围电路，就是reset晶振这些，先确保没问题，和同学对一下或者问优秀的学长学姐

	一些技术感悟：
	首先是原理图，先画芯片引脚，然后画外围最小系统，然后画一些转压，然后驱动板（其实强电弱电一定要分开，不过这次只有四个电机驱动，总电流不会超过1A，我就集成一下）
	记得排版好看一点，这在焊接或者交流的时候大有帮助。
	1.芯片如果外接rtc，布线不要打孔。
	2.12V5V这些稍微高压的增加线径，看对照表，估算电流
	3.画电路板的时候最好一次性画好（如果比较小，比如这次）
	4.一定要考虑好布局，布局的时候一个模块一个模块布局，不然你的布线会十分困难
	5.如果经济允许，可以自己去淘宝优信电子这类的网站配一套bom，然后自己买两本容阻本，不求人，这次芯片问题浪费我好多时间
	6.焊接时候，先焊接芯片，然后焊接转压，先测试转压是否好使，然后再通电给到芯片，不然就是点亮一颗芯片了
	这个文档可以不断更新，如果你们有什么感悟也可以不断添加
